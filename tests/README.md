# Tests

The workspace keeps Rust unit tests close to the modules they exercise.

Future integration, regression, property, and performance tests can be added here once the move engine and solvers exist.

Initial test focus:

- card encoding sanity
- state invariant validation
- stock/waste cycling
- move legality
- apply/undo symmetry
- hidden-tableau sampling correctness
- deterministic solver regressions
- paired benchmark reproducibility
